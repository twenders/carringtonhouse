# puzzles/

Puzzle data for the viewer.

- `puzzles.json` — manifest (chronological, oldest first; last entry is the site default). Each entry has `file`, `title`, `date`.
- `YY-MM-<name>.ipuz` — one file per puzzle. See [`../ipuz-format.html`](../ipuz-format.html) for the format.
- `example.ipuz` — minimal 3×3 example.
